# MFQ_Test_Plan（电商系统测试方案（MFQ 框架））

本仓库包含为电商系统设计的 MFQ 框架测试方案。

## 文件清单

| 文件 | 说明 |
|------|------|
| `MFQ_Test_Plan.xmind` | XMind 格式的测试方案（含 M-F-Q 三层结构及附录落地补充） |
| `MFQ_Test_Plan.md` | Markdown 格式的测试方案（与 XMind 内容一致，便于在线查看） |
| `Strategy_Explanation.md` | 1 页测试策略说明（优先级、取舍、风险驱动） |
| `README.md` | 本文件 |

## 内容覆盖

- **M 层**：业务流程模型、订单状态机、库存状态模型、外部交互模型、一致性边界
- **F 层**：核心主流程、库存同步专项测试（预占/扣减/恢复/幂等/乱序）、边界场景、API 校验
- **Q 层**：性能、可靠性、可观测性、安全性、可恢复性
- **发布准入/准出标准**
- **附录**：数据准备与环境要求、缺陷分级、测试工具、回滚机制
